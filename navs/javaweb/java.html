Java Web

O que é? Qual a funcionalidade? Onde é utilizado?

Na computação, Java Web Start foi uma estrutura desenvolvida pela Sun Microsystems que permite aos usuários iniciar o software aplicativo para a plataforma Java diretamente da Internet usando um navegador da web.
