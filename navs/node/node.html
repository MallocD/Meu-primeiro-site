Node Js

O que é? Qual a funcionalidade? Onde é utilizado?

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

Aqui está o botão que dá acesso as aulas

Matérias que servem de base para o estudo de Node Js:

  • HTTP
  • JSON
  • Promises
  • Arrow Functions
  • ES6
